# ch02_ownership

This lab practices Rust ownership, borrowing, references, and cloning with small
Focus Forge project examples.

Run it from the repository root:

```powershell
cargo run -p ch02_ownership
```

Verify it:

```powershell
cargo fmt --check
cargo check
cargo test -p ch02_ownership
cargo clippy --workspace --all-targets --all-features
```

Read the full guide in `chapters/02-ownership.md` and the assignment in
`assignments/ch02-ownership.md`.
