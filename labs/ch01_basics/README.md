# ch01_basics

This lab practices Rust's basic building blocks with a tiny Focus Forge project
summary.

Run it from the repository root:

```powershell
cargo run -p ch01_basics
```

Verify it:

```powershell
cargo fmt --check
cargo check
cargo test -p ch01_basics
cargo clippy --workspace --all-targets --all-features
```

Read the full guide in `chapters/01-basics.md` and the assignment in
`assignments/ch01-basics.md`.
