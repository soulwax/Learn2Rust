# ch00_setup

This is the first runnable Rust lab.

Run it from the repository root:

```powershell
cargo run -p ch00_setup
cargo run -p ch00_setup -- YourName
```

Verify it:

```powershell
cargo fmt --check
cargo check
cargo test
cargo clippy --workspace --all-targets --all-features
```

Read the full guide in `chapters/00-setup.md` and the assignment in `assignments/ch00-setup.md`.
